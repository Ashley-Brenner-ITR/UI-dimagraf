# Dudas e incoherencias - CU-002 Registrar y actualizar embarque / subcarpeta

## 1. Reversion de movimientos
- No queda definido si un embarque ya registrado puede editarse libremente o si necesita una regla de correccion/reversion.

## 2. Nomenclatura de subcarpeta
- El ERS habla de subcarpetas A, B, C, etc., pero no especifica si esa nomenclatura es obligatoria o solo ejemplificativa.

## 3. Relacion con validacion VAL
- No se explicita si la creacion del embarque dispara alguna validacion automatica adicional aparte de los controles de saldo.

## 4. Ajuste AS-IS
- En el proceso actual, una misma carpeta puede tener aperturas parciales (A, B, C...) y cada apertura conserva su propia factura y documentacion.
