# Dudas e incoherencias - CU-004 Gestionar documentacion del embarque

## 1. Ubicacion de los anexos
- El ERS contiene una contradiccion entre una solapa general de anexos y la visualizacion dentro de la instancia que los requiere.

## 2. Regla de visibilidad
- No se define si la visibilidad por rol se aplica a nivel de archivo completo, vista parcial o solo descarga.

## 3. Tipos de documento
- Se citan ejemplos de tipo de anexo, pero no queda cerrado el catalogo completo ni si es configurable.

## 4. Ajuste AS-IS
- En el proceso actual, la documentacion de cada apertura/subcarpeta se conserva asociada al embarque y forma parte del legajo operativo.
