# Dudas e incoherencias - CU-019 Comparar articulos (SKU + descripcion + cantidad)

## 1. Ponderacion de comparaciones
- No se define si el SKU domina totalmente sobre la descripcion cuando ambos difieren.

## 2. Tolerancia de cantidades
- No se fija el valor de tolerancia ni si cambia por proveedor o por tipo de articulo.

## 3. Ajuste AS-IS
- El control actual se hace articulo por articulo, revisando codigo, descripcion, cantidad, precio, formato, incoterm y condicion de pago de manera manual.
