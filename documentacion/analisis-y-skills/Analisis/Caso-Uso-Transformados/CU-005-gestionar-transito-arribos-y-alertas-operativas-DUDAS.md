# Dudas e incoherencias - CU-005 Gestionar transito, arribos y alertas operativas

## 1. Priorizacion de alertas
- No se detallan todas las reglas para decidir que alerta tiene prioridad cuando se solapan varios eventos.

## 2. Destinatarios
- No queda completamente definido si Tesoreria recibe todas las alertas o solo algunas categorias.

## 3. Fuente de actualizacion
- No se precisa si ETA y fechas reales pueden ser cargadas manualmente, importadas o ambas cosas.

## 4. Ajuste AS-IS
- El proceso actual combina planilla maestra, actualizacion manual semanal y consultas informales por correo o telefono para urgencias.
