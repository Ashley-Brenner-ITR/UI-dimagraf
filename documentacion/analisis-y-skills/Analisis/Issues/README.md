# Issues y seguimiento de correcciones

Este directorio reúne los documentos de issues, análisis y cambios asociados al proceso de importaciones.

## Estructura sugerida

- Documentos de issues: análisis de problemas reportados
- Documentos de entendimiento: causa raíz, impacto y plan de corrección
- Changelogs: historial de fixes
