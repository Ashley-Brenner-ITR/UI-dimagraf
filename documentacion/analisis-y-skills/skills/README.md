# Catálogo de Skills

Este directorio centraliza los skills usados para estructurar, refinar y transformar la documentación funcional del proceso de importaciones.

## Skills principales

- [refinamiento](refinamiento/SKILL.md): refina requerimientos ambiguos en definiciones funcionales más claras.
- [transformar-caso-uso](transformar-caso-uso/SKILL.md): convierte unidades refinadas en casos de uso estructurados.
- [generar-epica](generar-epica/SKILL.md): agrupa casos de uso relacionados en épicas funcionales.
- [agrupacion-epicas](agrupacion-epicas/SKILL.md): guía la agrupación lógica en entregables por dominio.
- [extraccion-unidades-refinables](extraccion-unidades-refinables/SKILL.md): identifica unidades refinables desde fuentes de análisis.
- [template-caso-uso](template-caso-uso/SKILL.md): define la estructura base de un caso de uso.
- [generar-issue-analisis](generar-issue-analisis/SKILL.md): produce documentación de análisis para issues.
- [generar-issue-fix](generar-issue-fix/SKILL.md): orienta la implementación de correcciones sobre issues.
- [generar-readme-general](generar-readme-general/SKILL.md): genera el README principal del repositorio.
- [crear-branch-pr](crear-branch-pr/SKILL.md): prepara y documenta la creación de branch y PR.
- [firma-itr](firma-itr/SKILL.md): incorpora la identidad institucional en los documentos.

## Uso recomendado

1. Iniciar desde la extracción de unidades refinables.
2. Refinar los requerimientos ambiguos.
3. Transformarlos en casos de uso.
4. Agruparlos en épicas funcionales.
5. Registrar issues y cambios cuando sea necesario.
